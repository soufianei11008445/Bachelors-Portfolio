Yellow-Legged-Hornets vs Bees Project

Introduction

This project aims to detect and track Yellow-Legged Hornets and Bees in still and video frames using AI tools.

Goals

  • Collect and Label Data: Extracted and labeled 104 images from existing datasets using Roboflow.
  • Data Augmentation: Applied three augmentations, increasing the dataset to 232 images, split into training, validation, and test sets.
  • Train Custom YOLO Model: Used transfer learning with a custom dataset, trained over 250 epochs using pre-trained weights and the Adam optimizer.
  • Model Evaluation: Evaluated model performance on the validation set using a confusion matrix, achieving higher average precision with the custom model.
  • Deploy on Streamlit: Developed a Streamlit app for real-time object detection from uploaded images and videos using the YOLOv8 model.

Conclusion

The project successfully developed a robust solution for detecting and tracking Yellow-Legged Hornets and Bees using a YOLOv8 model. Key achievements include effective data labeling, augmentation, model training, and deployment on Streamlit for user-friendly interaction. This showcases AI's potential in environmental monitoring and insect identification.
