Roses Image-Classification Using Different Models Project Summary

Introduction

This project classifies roses into seven categories using several trained models, with the best one deployed in a Streamlit app.

Goals

  • Collect and Label Data: Labeled 1,134 images from seven rose types.
  • Train FASTAI Models:
    • Simple Model: Used ResNet50, fine-tuned for three epochs.
    • AlexNet Model: Trained for 5 and 10 epochs, better performance at 10 epochs.
    • VGG Model: Trained for 5 and 10 epochs, VGG16 had the lowest error rate.
    • ResNet50: Optimized using learning rate finder and "fit_one_cycle", improved with FP16 training.
  • Train Keras Models (Extra):
    • Simple CNN Model: Accuracy improved from 0.29 (10 epochs) to 0.39 (20 epochs).
    • MobileNetV2 Model: Best Keras model, but not as good as the optimized Fastai models.
  • Deploy on Streamlit: Created an app for rose classification.

Conclusion

The project successfully found the best model for classifying roses. Highlights include effective data labeling and augmentation, training various models, with VGG16 performing the best, and deploying the model in a user-friendly Streamlit app.
